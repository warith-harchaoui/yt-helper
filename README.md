# yt-helper

**This repository is no longer maintained.**

It has moved to **[`youtube-helper`](https://github.com/warith-harchaoui/youtube-helper)**.

The Python module is now `youtube_helper`:

```python
import youtube_helper as yth
```

Install:

```bash
pip install --force-reinstall --no-cache-dir \
    git+https://github.com/warith-harchaoui/youtube-helper.git@v1.1.0
```

No functional changes — same surface, same behavior. The rename is purely a
PyPI naming-conflict resolution.

For history, all old `yt-helper` tags (`v0.1.0` … `v1.1.0`) remain readable
on this archived repository.
